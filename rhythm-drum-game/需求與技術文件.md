**中文名**: 节奏打鼓  
**英文名**: Rhythm Drum Idle / Air Drum Master  
**副标题**: 揮手擊鼓，節奏賺錢！

### 遊戲類型
- **主要類型**: 节奏遊戲 (Rhythm Game) + 放置遊戲 (Idle Game)

玩家通過 **空中揮手模擬敲鼓**，跟隨 **屏幕上的鼓譜 (Drum Chart)** 進行節奏判定。
- **互動模式**: 玩家親自敲鼓獲得高分數 + 收益加成
- **放置模式**: 自動鼓手持續演奏，24/7 賺錢
- **社交模式**: 錄製演奏分享，挑戰朋友

## 🎮 核心玩法

### 1. 基本玩法

#### **鼓譜系統 (Drum Chart)**
- 鼓譜以 **5 線譜** 形式呈現，包含 **4 種鼓聲**:
  | 鼓聲 | 顏色 | 音符位置 | 手勢判定 |
  |------|------|----------|----------|
  | **Bass Drum (大鼓)** | 🟥 紅色 | 最下方 | 双手同时敲击 (握拳) |
  | **Snare Drum (军鼓)** | 🟦 蓝色 | 下方 | 单手敲击 (揮手) |
  | **Hi-Hat ( 铲钹)** | 🟨 黃色 | 中間 | 快速揮手 (左右交替) |
  | **Crash Cymbal ( 锣)** | 🟪 紫色 | 最上方 | 双手上举 (张开手掌) |

- **判定時機**:
  - **Perfect** (±30ms): 3x 分數 + 連擊 (Combo) +1
  - **Good** (±60ms): 1.5x 分數 + 連擊 +0.5
  - **Bad** (±100ms): 0.5x 分數 + 連擊歸零
  - **Miss** (>100ms): 0x 分數 + 連擊歸零

#### **分數系統**
```
總分數 = (基礎分數 × 判定倍率) × (1 + 連擊加成 × 0.01) × (1 + 觀眾加成 × 0.001)
```
- **基礎分數**: 100 分/個音符
- **連擊加成**: 每 10 連擊 +1% 分數 (最高 +50%)
- **觀眾加成**: 觀眾數量 × 0.1% (最高 +1000%)

#### **放置機制 (Idle Mode)**
- **自動鼓手**: 遊戲启动后，系統自动生成一个 **AI 鼓手** 持續演奏
- **自动收益**: 每秒獲得 **基礎收益 × 鼓手等級 × 觀眾數量**
  - 基礎收益: $1/秒
  - 鼓手等級: 每級 +10% (最高 10 級)
  - 觀眾數量: 每 100 觀眾 +1% (最高 1000 觀眾)
- **離線收益**: 關閉遊戲后，仍可獲得 **50% 的離線收益** (最高 8 小時)

**技術方案 2: 自定义简单手势判定 (降级方案)**
```javascript
// 基于手部关键点的简单判定
function detectGesture(landmarks) {
  // 计算手指之间的距离
  const thumbIndexDist = distance(landmarks[4], landmarks[8]); // 拇指和食指
  const indexMiddleDist = distance(landmarks[8], landmarks[12]); // 食指和中指
  const palmWidth = distance(landmarks[0], landmarks[5]); // 手掌宽度
  
  // 判定逻辑
  if (thumbIndexDist < 0.1 * palmWidth && 
      indexMiddleDist < 0.1 * palmWidth) {
    return 'clap'; // 双手敲击 (大鼓)
  } else if (landmarks[8].y < landmarks[6].y) { // 食指比手腕高
    return 'snare'; // 单手敲击 (军鼓)
  } else if (landmarks[12].y < landmarks[10].y) { // 中指比手背高
    return 'hi-hat'; // 快速揮手 (铲钹)
  } else if (landmarks[0].y < landmarks[9].y) { // 手掌在手指上方
    return 'crash'; // 双手上举 (锣)
  }
  return null;
}
```

**手勢定義**:
| 鼓聲 | 手勢 | 判定邏輯 |
|------|------|----------|
| Bass Drum (大鼓) | **双手同时敲击** (握拳 + 快速向下) | 双手关键点 Y 坐标快速下降 |
| Snare Drum (军鼓) | **单手敲击** (食指向下) | 单手食指关键点 Y 坐标下降 |
| Hi-Hat (铲钹) | **快速左右揮手** | 手部 X 坐标快速变化 |
| Crash Cymbal (锣) | **双手上举** (张开手掌) | 双手关键点 Y 坐标高于头部 |

```javascript
// 自动生成随机鼓譜
function generateRandomChart(bpm, duration, difficulty) {
  const notes = [];
  const noteTypes = ['snare', 'bass', 'hi-hat', 'crash'];
  const weights = {
    easy: [0.5, 0.3, 0.15, 0.05], // 军鼓 50%, 大鼓 30%, 铲钹 15%, 锣 5%
    normal: [0.3, 0.3, 0.3, 0.1],
    hard: [0.2, 0.2, 0.4, 0.2]
  };
  
  const beatInterval = 60 / bpm; // 每拍的时间 (秒)
  const totalBeats = duration * bpm / 60;
  
  for (let i = 0; i < totalBeats; i++) {
    if (Math.random() > 0.7) continue; // 70% 几率空拍
    const noteType = weightedRandom(noteTypes, weights[difficulty]);
    notes.push({
      time: i * beatInterval,
      type: noteType,
      lane: noteTypes.indexOf(noteType)
    });
  }
  
  return { title: `Random Chart (${difficulty})`, bpm, duration, notes };
}
```

#### **E. 节奏判定系统 (Rhythm Judging System)**

**核心邏輯**:
```javascript
class RhythmJudge {
  constructor(bpm) {
    this.bpm = bpm;
    this.beatInterval = 60000 / bpm; // 每拍的毫秒数
    this.lastJudgedTime = 0;
    this.combo = 0;
    this.maxCombo = 0;
    this.score = 0;
  }
  
  // 判定玩家的敲击
  judge(gesture, timestamp) {
    const notes = this.getNotesInRange(timestamp, 100); // 获取 timestamp ±100ms 内的音符
    if (notes.length === 0) {
      this.resetCombo();
      return { result: 'Miss', score: 0 };
    }
    
    const closestNote = notes.reduce((prev, curr) => {
      return Math.abs(curr.time - timestamp) < Math.abs(prev.time - timestamp) ? curr : prev;
    });
    
    const diff = Math.abs(closestNote.time - timestamp);
    const judgeWindow = this.getJudgeWindow(closestNote.type);
    
    let result, scoreMultiplier;
    if (diff <= judgeWindow.perfect) {
      result = 'Perfect';
      scoreMultiplier = 3.0;
      this.combo++;
    } else if (diff <= judgeWindow.good) {
      result = 'Good';
      scoreMultiplier = 1.5;
      this.combo += 0.5;
    } else if (diff <= judgeWindow.bad) {
      result = 'Bad';
      scoreMultiplier = 0.5;
      this.resetCombo();
    } else {
      result = 'Miss';
      scoreMultiplier = 0;
      this.resetCombo();
    }
    
    const baseScore = 100;
    const comboBonus = 1 + (this.combo * 0.01); // 每 1 combo +1%
    const finalScore = baseScore * scoreMultiplier * comboBonus;
    
    this.score += finalScore;
    this.maxCombo = Math.max(this.maxCombo, this.combo);
    
    return { result, score: finalScore, combo: this.combo };
  }
  
  getJudgeWindow(type) {
    // 不同鼓声的判定窗口 (毫秒)
    const windows = {
      bass: { perfect: 50, good: 100, bad: 150 },
      snare: { perfect: 40, good: 80, bad: 120 },
      hiHat: { perfect: 30, good: 60, bad: 100 },
      crash: { perfect: 60, good: 120, bad: 180 }
    };
    return windows[type] || windows.snare;
  }
  
  resetCombo() {
    this.combo = 0;
  }
}
```

#### **F. 放置系统 (Idle System)**

```javascript
class IdleSystem {
  constructor() {
    this.coins = 0;
    this.audience = 100; // 初始觀眾
    this.drummers = [
      { type: 'street', level: 1, bonus: 1.0, autoEarn: 1 }
    ];
    this.lastActiveTime = Date.now();
    this.offlineEarnRate = 0.5; // 离线收益 50%
  }
  
  // 每秒自动赚钱
  update() {
    const now = Date.now();
    const deltaTime = (now - this.lastActiveTime) / 1000; // 转换为秒
    this.lastActiveTime = now;
    
    // 判断是否在线
    const isOnline = now - this.lastActiveTime < 5000; // 5 秒内有互动
    const earnRate = isOnline ? 1.0 : this.offlineEarnRate;
    
    // 计算总收益
    let totalEarn = 0;
    this.drummers.forEach(drummer => {
      totalEarn += drummer.autoEarn * drummer.bonus;
    });
    
    // 觀眾加成
    totalEarn *= (1 + this.audience * 0.001);
    
    // 最终收益
    this.coins += totalEarn * deltaTime * earnRate;
    
    // 觀眾变化
    if (isOnline) {
      this.audience = Math.min(this.audience + 0.1, this.getAudienceLimit());
    } else {
      this.audience = Math.max(this.audience - 0.2, 0);
    }
  }
  
  getAudienceLimit() {
    // 根据场地等级计算觀眾上限
    return 100 + this.drummers.length * 200;
  }
  
  // 雇佣新鼓手
  hireDrummer(type) {
    const drummerTypes = {
      'street': { bonus: 1.0, autoEarn: 1, cost: 100 },
      'jazz': { bonus: 1.2, autoEarn: 2, cost: 500 },
      'rock': { bonus: 1.3, autoEarn: 3, cost: 2000 },
      'metal': { bonus: 1.5, autoEarn: 5, cost: 10000 },
      'symphonic': { bonus: 2.0, autoEarn: 10, cost: 50000 }
    };
    
    if (!drummerTypes[type]) return false;
    if (this.coins < drummerTypes[type].cost) return false;
    
    this.coins -= drummerTypes[type].cost;
    this.drummers.push({
      type,
      level: 1,
      bonus: drummerTypes[type].bonus,
      autoEarn: drummerTypes[type].autoEarn
    });
    
    return true;
  }
  
  // 升级鼓手
  upgradeDrummer(index) {
    if (index >= this.drummers.length) return false;
    const drummer = this.drummers[index];
    const cost = drummer.level * 100;
    
    if (this.coins < cost) return false;
    
    this.coins -= cost;
    drummer.level++;
    drummer.bonus += 0.1; // 每级 +10% bonus
    
    return true;
  }
}
```

```javascript
    try {
      const data = JSON.parse(saveData);
      
      // 版本兼容性检查

        return this.migrate(data);
      }
      
      return data;

      return null;
    }
  }
  
  static migrate(oldData) {
    // 将来如果版本更新，在这里处理迁移逻辑
    return oldData;
  }
}
```

```javascript
class RecordingSystem {
  constructor() {
    this.mediaRecorder = null;
    this.recordedChunks = [];
    this.isRecording = false;
  }
  
  async startRecording(cameraStream, audioContext) {
    if (this.isRecording) return;
    
    // 合并摄像头和音频流
    const audioStream = this.createAudioStream(audioContext);
    const combinedStream = new MediaStream([
      ...cameraStream.getVideoTracks(),
      ...audioStream.getAudioTracks()
    ]);
    
    this.mediaRecorder = new MediaRecorder(combinedStream, {
      mimeType: 'video/webm;codecs=vp9,opus'
    });
    
    this.mediaRecorder.ondataavailable = (event) => {
      if (event.data.size > 0) {
        this.recordedChunks.push(event.data);
      }
    };
    
    this.mediaRecorder.onstop = () => {
      const blob = new Blob(this.recordedChunks, { type: 'video/webm' });
      this.recordedChunks = [];
      this.downloadRecording(blob);
    };
    
    this.mediaRecorder.start(1000); // 每 1 秒保存一个 chunk
    this.isRecording = true;
  }
  
  stopRecording() {
    if (!this.isRecording) return;
    
    this.mediaRecorder.stop();
    this.isRecording = false;
  }
  
  downloadRecording(blob) {
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = `rhythm-drum-performance-${Date.now()}.webm`;
    a.click();
    URL.revokeObjectURL(url);
  }
  
  createAudioStream(audioContext) {
    // 创建一个虚拟的音频流用于录制
    const destination = audioContext.createMediaStreamDestination();
    
    // 将所有音频节点连接到 destination
    // (需要修改音频播放逻辑，将音频输出到 destination)
    
    return new MediaStream([destination.stream.getAudioTracks()[0]]);
  }
}
```

```javascript
class DrumSynth {
  constructor() {
    this.audioContext = new (window.AudioContext || window.webkitAudioContext)();
  }
  
  play(type) {
    const now = this.audioContext.currentTime;
    
    switch(type) {
      case 'bass':
        this.playKick(now);
        break;
      case 'snare':
        this.playSnare(now);
        break;
      case 'hi-hat':
        this.playHiHat(now);
        break;
      case 'crash':
        this.playCrash(now);
        break;
    }
  }
  
  playKick(time) {
    // 大鼓音效 (正弦波 + 噪音)
    const oscillator = this.audioContext.createOscillator();
    const gain = this.audioContext.createGain();
    const filter = this.audioContext.createBiquadFilter();
    
    oscillator.type = 'sine';
    oscillator.frequency.setValueAtTime(60, time);
    
    gain.gain.setValueAtTime(0.5, time);
    gain.gain.exponentialRampToValueAtTime(0.001, time + 0.3);
    
    filter.type = 'lowpass';
    filter.frequency.setValueAtTime(300, time);
    filter.frequency.exponentialRampToValueAtTime(100, time + 0.3);
    
    oscillator.connect(filter);
    filter.connect(gain);
    gain.connect(this.audioContext.destination);
    
    oscillator.start(time);
    oscillator.stop(time + 0.3);
    
    // 添加一些噪音
    const noise = this.audioContext.createBufferSource();
    const noiseGain = this.audioContext.createGain();
    
    const buffer = this.audioContext.createBuffer(1, 0.3 * this.audioContext.sampleRate, this.audioContext.sampleRate);
    const data = buffer.getChannelData(0);
    for (let i = 0; i < data.length; i++) {
      data[i] = Math.random() * 2 - 1;
    }
    
    noise.buffer = buffer;
    noiseGain.gain.setValueAtTime(0.1, time);
    noiseGain.gain.exponentialRampToValueAtTime(0.001, time + 0.3);
    
    noise.connect(noiseGain);
    noiseGain.connect(this.audioContext.destination);
    noise.start(time);
  }
  
  playSnare(time) {
    // 军鼓音效 (噪音 + 頻率變化)
    const buffer = this.audioContext.createBuffer(1, 0.2 * this.audioContext.sampleRate, this.audioContext.sampleRate);
    const data = buffer.getChannelData(0);
    
    for (let i = 0; i < data.length; i++) {
      data[i] = Math.random() * 2 - 1;
    }
    
    const noise = this.audioContext.createBufferSource();
    const gain = this.audioContext.createGain();
    const filter = this.audioContext.createBiquadFilter();
    
    noise.buffer = buffer;
    
    gain.gain.setValueAtTime(0.3, time);
    gain.gain.exponentialRampToValueAtTime(0.001, time + 0.2);
    
    filter.type = 'highpass';
    filter.frequency.setValueAtTime(1000, time);
    filter.frequency.exponentialRampToValueAtTime(500, time + 0.2);
    
    noise.connect(filter);
    filter.connect(gain);
    gain.connect(this.audioContext.destination);
    
    noise.start(time);
    
    // 添加一个低频振动
    const oscillator = this.audioContext.createOscillator();
    const oscGain = this.audioContext.createGain();
    
    oscillator.type = 'sine';
    oscillator.frequency.setValueAtTime(200, time);
    oscillator.frequency.exponentialRampToValueAtTime(100, time + 0.2);
    
    oscGain.gain.setValueAtTime(0.2, time);
    oscGain.gain.exponentialRampToValueAtTime(0.001, time + 0.2);
    
    oscillator.connect(oscGain);
    oscGain.connect(this.audioContext.destination);
    oscillator.start(time);
    oscillator.stop(time + 0.2);
  }
  
  playHiHat(time) {
    // 铲钹音效 (高频噪音)
    const buffer = this.audioContext.createBuffer(1, 0.1 * this.audioContext.sampleRate, this.audioContext.sampleRate);
    const data = buffer.getChannelData(0);
    
    for (let i = 0; i < data.length; i++) {
      data[i] = Math.random() * 2 - 1;
    }
    
    const noise = this.audioContext.createBufferSource();
    const gain = this.audioContext.createGain();
    const filter = this.audioContext.createBiquadFilter();
    
    noise.buffer = buffer;
    
    gain.gain.setValueAtTime(0.2, time);
    gain.gain.exponentialRampToValueAtTime(0.001, time + 0.1);
    
    filter.type = 'highpass';
    filter.frequency.setValueAtTime(5000, time);
    
    noise.connect(filter);
    filter.connect(gain);
    gain.connect(this.audioContext.destination);
    
    noise.start(time);
  }
  
  playCrash(time) {
    // 锣音效 (长衰减噪音)
    const buffer = this.audioContext.createBuffer(1, 0.5 * this.audioContext.sampleRate, this.audioContext.sampleRate);
    const data = buffer.getChannelData(0);
    
    for (let i = 0; i < data.length; i++) {
      data[i] = Math.random() * 2 - 1;
    }
    
    const noise = this.audioContext.createBufferSource();
    const gain = this.audioContext.createGain();
    const filter = this.audioContext.createBiquadFilter();
    
    noise.buffer = buffer;
    
    gain.gain.setValueAtTime(0.4, time);
    gain.gain.exponentialRampToValueAtTime(0.001, time + 0.5);
    
    filter.type = 'lowpass';
    filter.frequency.setValueAtTime(3000, time);
    filter.frequency.exponentialRampToValueAtTime(100, time + 0.5);
    
    noise.connect(filter);
    filter.connect(gain);
    gain.connect(this.audioContext.destination);
    
    noise.start(time);
  }
}
```

```javascript
    this.playingCount = 0;
    this.queue = [];
  }
  
  async init() {
    const soundFiles = {
      bass: 'assets/sounds/bass.mp3',
      snare: 'assets/sounds/snare.mp3',
      hiHat: 'assets/sounds/hi-hat.mp3',
      crash: 'assets/sounds/crash.mp3'
    };
    
    for (const [type, url] of Object.entries(soundFiles)) {
      const audio = new Audio(url);
      audio.preload = 'auto';
      this.sounds[type] = audio;
    }
  }
  
  play(type) {
    if (this.playingCount >= this.maxConcurrent) {
      // 如果超过最大并发数，加入队列
      this.queue.push(type);
      return;
    }
    
    const audio = this.sounds[type];
    if (!audio) return;
    
    audio.currentTime = 0;
    audio.play().catch(err => {
      console.error('播放失败:', err);
    });
    
    this.playingCount++;
    
    audio.onended = () => {
      this.playingCount--;
      // 处理队列
      if (this.queue.length > 0) {
        const nextType = this.queue.shift();
        this.play(nextType);
      }
    };
  }
}
```
